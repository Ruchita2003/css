Que-16 What are the advantages and disadvantages of using external style sheets?


Ans-Advantages of External Style Sheets:
Reusability: With external style sheets, you can define styles once and reuse them across multiple HTML files.
Maintainability: Keeping CSS code separate in external files improves code organization and maintainability.
Efficiency: External style sheets can be cached by the browser, which means once a style sheet is loaded, subsequent page loads can use the locally stored version.
Browser Compatibility: External style sheets provide a consistent way to apply styles across different browsers.
Disadvantages of External Style Sheets:
Increased HTTP Requests: Using external style sheets requires an additional HTTP request to fetch the CSS file.
Dependency: External style sheets rely on the availability and correct linking of the CSS file.
Complexity for Small Websites: For small websites with a limited number of pages and simple styling requirements, external style sheets might introduce unnecessary complexity.
Performance Impact: In certain cases, if an external style sheet is large and contains many unused styles, it can impact performance.

Que-17 What is the meaning of the CSS selector?


Ans-a selector is a pattern or rule used to select HTML elements to which styles should be applied.
Selectors determine which elements in an HTML document will receive the specified styles defined in the CSS code.
CSS selectors are based on different criteria such as element type, class, ID, attributes, and hierarchical relationships.


Que- 18: What are the media types allowed by CSS?


Ans-In media queries, the @media rule is used to apply various styles to different media types/devices. Many things can be checked with media queries, including the width and height of the viewport, as well as the width and height of the device..


Example:

<style tyle = "text/css">
@media print {
body { font-size: 10pt }
}
@media screen {
body { font-size: 12pt }
}
@media screen, print {
body { line-height: 1.2 }
}
</style>
